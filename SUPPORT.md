# Support

## Issues

Report bugs and request features here:

- https://github.com/JacobTheJacobs/vscode-supergrok/issues

## Before Filing

- Confirm the Grok CLI works in a terminal.
- Confirm you are signed in to the Grok CLI.
- Include the VS Code version, OS, and Grok CLI version.
- Include the extension logs if the problem involves startup, sessions, or media.
